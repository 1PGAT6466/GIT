### 九、风险评估

- 🟢 低：本次修改仅涉及 CPU 采样异步化和前端 UI 微调

### 十、后续建议

3. 【中期】修复 `on_event` deprecation → 改为 lifespan
4. 【中期】修复 `regex` deprecation → 改为 `pattern`
